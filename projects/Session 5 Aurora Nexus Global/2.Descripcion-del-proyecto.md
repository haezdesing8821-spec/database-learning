Dentro de este proyecto desarrollaré y documentaré el proceso comleto del diseño, modelado normalizacion y adaptación de una base de datos desde un modelo `entidad-relación` a un `modelo relacional`, posterior a eso también estaremos documentando todo el proceso de aplicación a SQL dentro de un `DBMS`A NIVEL DE `CÓDIGO Y LOGISTICA` RESPETANDO LOS ESTANDARES UNIVERSALES *`ISO y ANSI` COMO DARLE MANTENIMIENTO A UNA BASE DE DATOS RELACIONAL EN SQL, PRACTICAR LOS FUNDAMENTOS Y LOS CONCEPTOS AVANZADOS DE LENGUAJE `SQL`, COMO SACAR INFORMACIÓN DE UNA BASE DE DATOS Y MANIPULAR ESOS DATOS CON DIVERSAS HERRAMIENTAS PARA REALIZAR REPORTES EFICIENTES, PERO PASO A PASO.

EN LA SIGUIENTE SESIÓN TRABAJAREMOS CON UN ENUNCIADO QUE TENDREMOS QUE DISEÑAR Y NORMALIZAR A UN MODELO `ENTIDAD-RELACIÓN` APRENDIENDO TODO RELACIONADO CON `MER` PARA QUE PUEDAN ENTENDER EL FUNCIONAMIENTO Y LECTURA CONCEPTUAL DE UN MODELO *`ENTIDAD-RELACIÓN` DE UNA BASE DE DATOS PARA LUEGO PASARLO Y/O CONVERTIRLO A UN MODELO RELACIONAL.
